# McLittleLang

**McLittleLang** is a lightweight utility that allows you to create Minecraft structures using a Logo-like language. It provides an intuitive way to design and generate complex structures within the Minecraft world, leveraging the simplicity and elegance of Logo programming concepts.

## Features

- **Logo-like Syntax**: Use simple commands to move, rotate, and place blocks, similar to the Logo programming language.
- **Customizable Structures**: Create custom structures using an easy-to-understand scripting language.
- **Interactive REPL**: Test and modify your scripts in real-time with an interactive command-line interface.
- **Integration with Minecraft**: Easily export your designs to Minecraft and bring your structures to life.
- **Cross-Platform**: Available on multiple platforms including Windows, macOS, and Linux.
